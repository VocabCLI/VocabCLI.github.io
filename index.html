VocabCLI

A lightweight Command Line Interface that allows users to look up word definitions, examples, synonyms and antonyms directly via the command line.
Powered with several utility based commands our CLI offers rapid and robust Knowledge Base capabilities like Flashcards, Tagging, Word Management, Graph Reporting, Bulk import and export of word lists and is a definitive software for linguaphiles.
